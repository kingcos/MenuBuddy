功能亮点

🤖

AI 反应

接入任意 OpenAI 兼容 API（DeepSeek、OpenAI、Ollama 等），根据伙伴的性格属性生成个性化反应。

📊

系统监控

实时响应 CPU、内存、网络、磁盘 I/O 和电池状态。弹窗中显示实时指标条。

⚡

脚本触发器

将可执行脚本放入 ~/.menubuddy/triggers/ 即可生效 —— 股票、天气、CI 状态，随你发挥。

🎨

动画精灵

3 帧待机动画，眨眼循环，抚摸时爱心爆发。每个物种都有独特的 ASCII 造型。

💬

场景感知台词

检测你正在使用的应用（IDE、浏览器、终端）并发表评论。支持勿扰时段。

🌐

双语支持

完整的英文和简体中文支持。300+ 本地化字符串，应用内切换语言。

内置系统触发器

你的伙伴会监控 Mac 状态并实时做出反应。

指标阈值指示器反应
CPU>70%🔥压力眼 (x)
内存>85% 占用🧠波浪眼 (~)
网络>5 MB/s⚡火箭心情
磁盘 I/O>50 MB/s💾圆眼 (o)
电池<20%🪫小点眼 (.)

自定义脚本触发器

将可执行脚本放入 ~/.menubuddy/triggers/ 即可：

#!/bin/bash # ~/.menubuddy/triggers/stock.sh echo '{"interval":60,"trigger":{"indicator":"📈","quips":["涨了!"]},"metrics":[{"label":"AAPL","value":"$254","trend":"↑"}]}'
