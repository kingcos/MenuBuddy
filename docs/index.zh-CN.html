你的伙伴，你的 Mac

伙伴由 Mac 硬件 UUID 确定性生成。同一台机器，永远是同一个伙伴。个性属性决定了它的说话方式和反应。

  • 5 个性格属性影响每一句话
  • 接入任意 OpenAI 兼容 API 驱动 AI 对话
  • 弹出面板实时展示系统指标
  • 菜单栏双行显示：表情 + 话语
  • 点击抚摸，右键打开菜单
  • 感知当前应用 —— 知道你在做什么

功能亮点

让你的菜单栏伙伴真正「活」起来。

🤖

AI 反应

接入任意 OpenAI 兼容 API（DeepSeek、OpenAI、Ollama 等），伙伴的性格属性会影响每一句 AI 生成的回复。

✨

AI 优化触发语句

一键开关，让 AI 改写系统事件消息。CPU 过高时，它不会只说「好热」——而是用自己的性格来表达。

📊

系统监控

实时响应 CPU、内存、网络、磁盘 I/O 和电池状态。弹窗内显示实时指标条和趋势箭头。

⚡

脚本触发器

将可执行脚本放入 ~/.menubuddy/triggers/ 即可 —— 股票、天气、CI 状态，随你发挥。

🎨

动画精灵

3 帧待机动画、眨眼循环、抚摸时爱心爆发。每个物种都有独特的 ASCII 造型和菜单栏表情。

💬

场景感知

检测你正在使用的应用（IDE、浏览器、终端、音乐）并发表评论。支持勿扰时段设置。

📏

紧凑菜单栏

双行显示：上面是表情，下面是话语，节省横向空间。打开弹窗时自动隐藏话语文字。

🌏

双语支持

完整的英文和简体中文。300+ 本地化字符串，应用内切换语言。AI 回复自动匹配上下文语言。

🔒

隐私优先

无账号、无遥测、不联网（除非你开启 AI）。一切数据留在本地。已签名并公证。

触发器系统

内置系统监控 + 可扩展脚本。你的伙伴实时响应 Mac 状态。

指标阈值指示器反应
CPU> 70%🔥压力眼 (x)
内存> 85%🧠波浪眼 (~)
网络> 5 MB/s⚡火箭心情 🚀
磁盘 I/O> 50 MB/s💾圆眼 (o)
电池< 20%🪫小点眼 (.)

自定义脚本触发器

将可执行脚本放入 ~/.menubuddy/triggers/ 即可：

#!/bin/bash # ~/.menubuddy/triggers/stock.sh echo '{"interval":60,"trigger":{"indicator":"📈","quips":["涨了!"]},"metrics":[{"label":"AAPL","value":"$254","trend":"↑"}]}'
