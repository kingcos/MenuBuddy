Your Buddy, Your Mac

Generated deterministically from your Mac's hardware UUID. Same machine, same buddy — always. Personality stats drive how it talks, reacts, and behaves.

  • 5 personality stats shape every reaction
  • AI-powered speech via any OpenAI-compatible API
  • Real-time system metrics in the popover
  • Two-line menu bar: face + speech
  • Click to pet, right-click for menu
  • Context-aware — knows what app you're using

Features

Everything you need for a companion that feels alive.

🤖

AI Reactions

Connect any OpenAI-compatible API — DeepSeek, OpenAI, Ollama, and more. Your buddy's personality stats shape every AI-generated response.

✨

AI-Enhanced Triggers

Toggle AI rephrasing of system event messages. CPU high? Your buddy won't just say “hot” — it'll say something in character.

📊

System Monitor

Real-time reactions to CPU, memory, network, disk I/O, and battery. Live metric strip in the popover with trend arrows.

⚡

Script Triggers

Drop any executable into ~/.menubuddy/triggers/ — stock prices, weather, CI builds. JSON output, zero framework.

🎨

Animated Sprites

3-frame idle animation with blink cycles. Heart burst on pet. Each species has unique ASCII art and menu bar face.

💬

Context-Aware

Detects which app you're using — IDE, browser, terminal, music — and comments on it. Do Not Disturb for quiet hours.

📏

Compact Menu Bar

Two-line display: face on top, speech below. Saves horizontal space. Auto-hides speech text when the popover is open.

🌏

Bilingual

Full English and Simplified Chinese. 300+ localized strings with in-app language switcher. AI replies match context language.

🔒

Privacy-First

No account, no telemetry, no network unless you enable AI. Everything stays on your Mac. Signed & notarized.

Custom Script Triggers

Drop an executable into ~/.menubuddy/triggers/ and it just works:

#!/bin/bash # ~/.menubuddy/triggers/stock.sh echo '{"interval":60,"trigger":{"indicator":"📈","quips":["AAPL is up!"]},"metrics":[{"label":"AAPL","value":"$254","trend":"↑"}]}'
