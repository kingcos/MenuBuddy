Features

🤖

AI Reactions

Connect any OpenAI-compatible API (DeepSeek, OpenAI, Ollama) for personality-driven reactions. Your buddy's stats shape how it talks.

📊

System Monitor

Reacts to CPU, memory, network, disk I/O, and battery. See live metrics in the popover status strip.

⚡

Script Triggers

Drop any executable into ~/.menubuddy/triggers/ — stock prices, weather, CI status. No coding framework needed.

🎨

Animated Sprites

3-frame idle animation, blink cycles, heart burst on pet. Each species has unique ASCII art.

💬

Context-Aware Quips

Detects which app you're using (IDE, browser, terminal) and comments on it. Do Not Disturb mode for quiet hours.

🌐

Bilingual

Full English and Simplified Chinese support. 300+ localized strings with in-app language switcher.

Built-in System Triggers

Your buddy monitors your Mac and reacts in real time.

MetricThresholdIndicatorReaction
CPU>70%🔥Stress eyes (x)
Memory>85% used🧠Squiggly eyes (~)
Network>5 MB/s⚡Rocket mood
Disk I/O>50 MB/s💾Wide eyes (o)
Battery<20%🪫Tiny dot eyes (.)

Custom Script Triggers

Drop an executable into ~/.menubuddy/triggers/ and it just works:

#!/bin/bash # ~/.menubuddy/triggers/stock.sh echo '{"interval":60,"trigger":{"indicator":"📈","quips":["AAPL is up!"]},"metrics":[{"label":"AAPL","value":"$254","trend":"↑"}]}'
